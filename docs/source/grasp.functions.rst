grasp.functions module
======================

.. note::

   This module currently exposes only
   :class:`grasp.functions.CartesianConversion`. The older
   :class:`AngularSeparation`, :class:`GcDistance` and
   :class:`RadialDistance2D` classes were removed in the Phase 1 cleanup
   because their derivations duplicate utilities already provided by
   :mod:`astropy.coordinates`. Use those for new code.
